# DM-Phenomenology

# Strong Gravitational Lensing Pipeline Validation (Synthetic Data)

## Overview

This project presents a computational pipeline for modeling strong gravitational lensing systems using forward modeling and Bayesian inference. The pipeline is validated using synthetic data to ensure its ability to recover lens and source parameters reliably.

The work focuses on building an end-to-end framework that connects physical lens modeling with numerical optimization techniques.

---

## Objectives

- Develop a forward model for strong gravitational lensing
- Simulate synthetic lensed images using known parameters
- Perform parameter inference using MCMC sampling
- Validate the pipeline by comparing recovered parameters with ground truth

---

## Methodology

### 1. Lens Model
- Singular Isothermal Ellipsoid (SIE) approximation
- Includes external shear
- Parameters:
  - Einstein radius (θ_E)
  - Axis ratio (q)
  - Shear (γ, φ)

---

### 2. Source Model
- Elliptical Sersic profile
- Parameters:
  - Effective radius (R_e)
  - Axis ratio (q_src)
  - Orientation (φ_src)
  - Source position (sx, sy)

---

### 3. Forward Modeling

The observed image is generated by:

1. Mapping image plane → source plane using lens equation
2. Evaluating source brightness distribution
3. Convolution with PSF
4. Adding noise (for synthetic validation)

---

### 4. Inference Method

- Bayesian parameter estimation using Metropolis-Hastings MCMC
- Likelihood based on pixel-wise residuals between model and data
- Priors imposed to ensure physical solutions

---

## Synthetic Validation

### Setup

- Synthetic images generated using known ("ground truth") parameters
- Gaussian noise added to simulate observational conditions

### Goal

To test whether the pipeline can:

- Recover input parameters
- Reconstruct lensed arcs accurately
- Produce meaningful posterior distributions

---

## Results

- The pipeline successfully recovers key parameters such as:
  - Einstein radius (θ_E)
  - Shear (γ)
  - Source size (R_e)
- Residual maps show reduced structure after fitting
- MCMC chains converge with reasonable acceptance rates (~0.2–0.4)

---

## Key Insights

- Proper likelihood design is critical for stable inference
- Parameter degeneracies (e.g., shear vs ellipticity) affect convergence
- Masking and noise modeling significantly impact reconstruction quality
